# SAHAYAK

AccessibleFlow: AI-Powered Accessibility Platform
An open-source, mobile-first web platform designed to empower the 26 million differently-abled individuals in India by providing real-time AI assistive tools directly in their browsers.

🚀 Overview
AccessibleFlow bridges the digital divide by offering a suite of AI-powered features tailored for users with visual, auditory, speech, and learning disabilities. By leveraging a mobile-web architecture, we ensure that powerful accessibility tools are accessible on low-cost Android devices without the need for high-end hardware or complex app installations.

🔑 Key Features
ISL-to-Speech/Text: Real-time Indian Sign Language recognition using Convolutional Neural Networks (CNN) to convert hand gestures into actionable text and spoken audio.

Visual Assistant: A scene description engine powered by Transformer models that processes camera input to describe surroundings, helping visually impaired users navigate their environment.

Empowered Speech: Custom acoustic models optimized for users with speech impediments, providing highly accurate speech-to-text transcription.

Cognitive Support: A text-simplification engine that breaks down complex language into easy-to-read formats for users with learning disabilities.

Seamless Integration: Native-like integration capabilities with daily-use applications like WhatsApp and Gmail to improve user productivity and communication.

🛠 Technical Architecture
Our stack is optimized for performance on resource-constrained devices, utilizing lightweight models and efficient web-based inference.

🎯 Impact
With an estimated 26 million differently-abled individuals in India, many currently lack affordable, specialized technology. This project aims to:

Remove barriers to communication for the hearing and speech-impaired.

Increase independent mobility for the visually impaired.

Promote inclusive digital literacy through intuitive UI/UX design.
